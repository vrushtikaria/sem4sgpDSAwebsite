    • Finding number of digits in a number

    1. Iterative Solution:

      2. int digitcounter(long n) {
      int count = 0;
      while (n != 0) {
      n = n / 10;
      count++;
      }
      return count;
      }

    2. Recursive Solution:

      3. int digitcounter(long n) {
      if (n == 0)
      return 0;
      return 1 + digitcounter(n / 10);
      }

    3. Logarithmic Solution:

      4. int digitcounter(long n) {
      return floor ( log10 (n) + 1);
      }

    Arithmetic and Geometric Progressions (AP, GP)

    • An arithmetic progression is a sequence of numbers such that the difference of any two successive members is a constant called the common difference.

      • We denote by d the common difference.

      • By aₙ we denote the n-th term of an arithmetic progression.

        • aₙ = a₁ + (n-1)d


      • By Sₙ we denote the sum of the first n elements of an arithmetic series.

        • sₙ = n/2(2a₁ + (n-1)d)


    • A geometric progression is a sequence of numbers such that the quotient of any two successive members of the sequence is a constant called the common ratio of the sequence.

      • The geometric progression can be written as:

        • ar⁰=a, ar¹=ar, ar², ar³, …

      • where r ≠ 0, r is the common ratio and a is the first term.
      • Formula for the n-th term can be defined as:

        • aₙ = a⋅rⁿ⁻¹

      • Formula for the sum of the first n numbers of a geometric series:

        • Sₙ = a (1-rⁿ)/(1-r)

    Quadratic Equations

      ax² + bx + c = 0

    • Taking input from the user of the quadratic equation means taking a, b and c input from the user.

    • We can have maximum of two roots in a quadratic equation.The nature of roots depends upon discriminant (D):

      • D = b²-4*a*c

    • If determinant is greater than 0, the roots are real and different.
    • If determinant is equal to 0, the roots are real and equal.
    • If determinant is less than 0, the roots are complex and different.
